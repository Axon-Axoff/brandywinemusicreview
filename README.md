Brandywine Music Review is a music sharing and rating web app built with PHP, MySQL, and Javascript.

Brandywine Music Review allows users to post albums and songs and comment on and rate them. Spotify, 
Youtube, and Bandcamp links will play in the apps music player and can be included for reference.
Users can rate and comment on other user's entries and all ratings are averaged by rating and 
weighted rating. Users are notified if any of their entries are rated or commented on. Users can also 
message each other through members page.

To install drop entire contents into Apache public_html directory and import musicdb.sql with mysql or
phpmyadmin. This will write an empty database structure. Create and admin account in users via mysql or 
phpmyadmin. This account will be able to promote regular user accounts to admin accounts if needed.
